# Eventio

**Eventio** är en skräddarsydd webbplattform för utomhusevent med fokus på schemaläggning, deltagarkommunikation och smidig administration. Verktyget är utvecklat för att göra evenemangsplanering enklare och mer tillgänglig – direkt i mobilen eller datorn.

🌐 [eventio.se](https://eventio.se)

## 🚀 Funktioner

- 📅 **Schemaläggning i realtid**  
  Visa och uppdatera schemat direkt för deltagare och arrangörer.

- 🗣️ **Kommunikationskanaler**  
  Inbyggda meddelanden, push-notiser och viktiga aviseringar.

- 📍 **Platsbaserade aktiviteter**  
  Visa aktiviteter på karta eller filtrera per område.

- 📲 **Progressive Web App (PWA)**  
  Offline-stöd och installation på hemskärm – fungerar som en app!
